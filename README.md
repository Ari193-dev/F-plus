# F+ 
This DSL is made in C and is stack based. If you choose the open source version you must have GCC to compile it and you will be able to see all the code. If you choose the compiled version, you can use the Windows executable (x86-64) to run the programs created on this DSL. 
Example of use: 
PUSH, 15, 
INPUT, 
MUL, 
PRINT_STACK, 
RESET 
That code puts 15 on the stack, makes the user put another number, multiplies the 15 and what is put by the user and saves that result on the stack, (removing all the 15 and what is put by the user from the stack), print it and start again.
